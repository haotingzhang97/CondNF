This repository contains code for my MSc Machine Learning project at UCL. 

The implementation of U-Net and pix2pix are modified from the original authors of pix2pix:

- Image-to-Image Translation with Conditional Adversarial Networks.<br>
[Phillip Isola](https://people.eecs.berkeley.edu/~isola), [Jun-Yan Zhu](https://people.eecs.berkeley.edu/~junyanz), [Tinghui Zhou](https://people.eecs.berkeley.edu/~tinghuiz), [Alexei A. Efros](https://people.eecs.berkeley.edu/~efros). In CVPR 2017.<br>
[[arxiv]](https://arxiv.org/abs/1611.07004); [[repo]](https://github.com/junyanz/pytorch-CycleGAN-and-pix2pix)

The implementation of MSGAN is modified from its original authors:

- Mode Seeking Generative Adversarial Networks for Diverse Image Synthesis.<br>
[Qi Mao](https://sites.google.com/view/qi-mao/)\*, [Hsin-Ying Lee](http://vllab.ucmerced.edu/hylee/)\*, [Hung-Yu Tseng](https://sites.google.com/site/hytseng0509/)\*, [Siwei Ma](https://scholar.google.com/citations?user=y3YqlaUAAAAJ&hl=zh-CN), and [Ming-Hsuan Yang](http://faculty.ucmerced.edu/mhyang/). In CVPR, 2019 (* equal contribution)<br>
[[arxiv]](https://arxiv.org/abs/1903.05628); [[repo]](https://github.com/HelenMao/MSGAN)

The implementation of c-Glow is modified from its original authors:

- Structured Output Learning with Conditional Generative Flows.<br>
You Lu, Bert Huang. In AAAI, 2020<br>
[[arxiv]](https://arxiv.org/pdf/1905.13288); [[repo]](https://github.com/yolu1055/conditional-glow)

The implementations of Probabilistic U-Net and LIDC dataloader are modified from the following github repository:

- A Probabilistic U-Net for segmentation of ambiguous images implemented in PyTorch [[repo]](https://github.com/stefanknegt/Probabilistic-Unet-Pytorch)
